Babel 에 대해서 학습한다.

        Babel 이란 JS 컴파일러(=번역기)다.
            ES6 를 ES5 로 바꾸어 주는
            JSX 를 ES로 바꾸어 주는

        온라인 babel 컴파일러 : https://babeljs.io/repl/

        바벨은 JS 컴파일러
        ES2016 작성코드를 ES5로 변경해준다.
        React 의 JSX(리액트 화면을 구성하는 언어) 를 컴파일 해주기도한다.

        


        변수선언:
            ES5 : var        : 함수 스코팅
            ES6 : let, const : 블락 스코핑
        함수선언:
            ES5 : function(){}
            ES6 : ()=>{ }
